Trend Tracker
Trend Tracker es una app pensada para creadores de contenido que quieren detectar rapidamente audios, retos, tematicas y formatos en crecimiento para TikTok, Instagram Reels y YouTube Shorts.
La aplicacion permite explorar tendencias por nicho, ver audios virales, analizar desafios actuales y obtener ideas de contenido con hooks listos para usar en los primeros segundos del video.
El objetivo principal es ayudar a los creadores a encontrar oportunidades antes de que una tendencia se sature, facilitando la creacion de contenido relevante, rapido y con mayor potencial de alcance.
Funcionalidades principales
Dashboard con tendencias destacadas.
Busqueda por palabra clave o nicho.
Listado de audios virales con metricas de crecimiento.
Retos y desafios con nivel de competencia.
Ideas de contenido y hooks para videos cortos.
Estado
Proyecto en etapa inicial de MVP.
